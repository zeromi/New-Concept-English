- [x] **NCE1**相当于初中英语水平,大部分文章为口语对话型,生动幽默,培养兴趣为主
- [x] **NCE2**相当于高中英语水平,文章为口语和叙事型,涵盖了英语主要语法,能够打牢基础
- [x] **NCE3**相当于大学四级水平,文章为叙事和议论型,句子的长度和难度都有加大,对英语水平有质的提升
- [ ] **NCE4**中涵盖新概念四的全部语法,相当于大学六级水平
